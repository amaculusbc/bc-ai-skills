# Article Writer

Drafts publish-ready articles from a brief, in property voice, compliance-safe.

## Use it
**Live Gem:** https://gemini.google.com/gem/REPLACE_WITH_SHARE_LINK

Use the shared Gem above. Don't make your own copy unless you're changing the skill.

## Recreate / update the Gem (maintainers)
1. New Gem (or edit existing) → paste `gem.md` as instructions.
2. Attach the `uses:` files from `meta.yaml` + anything in `knowledge/`.
3. Bump `version`, update `gem_url` if the link changed, run the registry script.
4. Test against `examples/test-cases.md`.
